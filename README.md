# Parametric Hierarchical Matrix Approximations to Kernel Matrices
This repository contains Python code for computing parametric hiearchical matrix approximations to kernel matrices. It accompanies the paper


The results generated in the [folder](https://github.com/awkhan3/ParametricHierarchicalMatrices/tree/master/Logs) were generated by this repository on the Hazel Cluster at North Carolina State University.

## Requirements
The Python code requires the following packages to generate the tables.
1. Numpy
2. Scipy
3. Numba
4. Teneva

To run all of the experiments, simply clone this directory,  then run the command 'python3 run_test.py'.
